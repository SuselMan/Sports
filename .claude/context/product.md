# Product: KekTrack (Sports)

## Vision
Минималистичный трекер силовых тренировок. Пользователь создаёт свои упражнения, указывает целевые группы мышц, записывает подходы и видит прогресс на карте мышц.

## Target Audience
Люди, занимающиеся силовыми тренировками, которым нужен простой и быстрый инструмент для записи подходов без лишней сложности.

## Core Value Proposition
- **Простота**: никаких готовых программ, шаблонов, социальных функций — только твои упражнения и твой прогресс
- **Визуализация**: карта мышц показывает, что тренировано сегодня
- **Offline-first**: работает без интернета, синхронизируется при подключении
- **PWA**: устанавливается на телефон как нативное приложение

## Key Features (implemented)
1. **Карта мышц** — SVG-визуализация тела с подсветкой тренированных мышц за выбранный период
2. **Упражнения** — CRUD, два типа: REPS (повторения) и TIME (время), привязка к группам мышц
3. **Записи тренировок** — подходы с весом, RPE, временем отдыха, заметками
4. **Метрики** — произвольные числовые показатели (вес тела, пульс, калории и т.д.)
5. **Статистика** — графики прогресса по упражнениям (макс повторения, e1RM, подходы/день) и метрикам
6. **Offline-first** — IndexedDB + incremental sync с сервером
7. **Интернационализация** — 10 языков (en, ru, es, fr, pt, zh, hi, ar, bn)
8. **Темы** — светлая и тёмная

## Key Features (not yet implemented / from spec)
- **Introducing screen** — онбординг при первом входе (пол, возраст)
- **Генерация карты мышц по полу** (мужской/женский силуэт)
