# Training Days Chips — визуализация регулярности тренировок

**Status**: DONE
**Priority**: P1
**Category**: ux

## Description

Добавить горизонтальную полоску чипов под quick filters (Today/Week/Month/Year) на Home странице. Чипы показывают ленту периодов, где акцентным цветом подсвечены периоды с тренировками (exercise records, не метрик). Клик по чипу переключает date range на соответствующий период.

## Гранулярность чипов

Гранулярность чипа соответствует выбранному quick filter:

- **Day** → чипы = дни (~14 дней вокруг выбранного), формат: "Пн 7", "Вт 8"
- **Week** → чипы = недели (~12 недель), формат: "Нед 14", "Нед 15"
- **Month** → чипы = месяцы (12 месяцев), формат: "Янв", "Фев"
- **Year** → чипы = годы (показывать больше лет чем есть данных), формат: "2024", "2025"
- **Custom** → определяется по длине диапазона:
  - до 14 дней → чипы = дни
  - до 90 дней → чипы = недели
  - больше → чипы = месяцы

## Поведение

- Акцентный цвет — период содержит хотя бы одну запись упражнения (не archived)
- Обычный цвет — нет тренировок в этом периоде (визуально видны "дырки")
- Клик по чипу — сдвигает date range на этот период (с сохранением длительности)
- Текущий выбранный период выделен дополнительно (рамка или другой стиль)
- Автоскролл к текущему выбранному периоду
- Горизонтальный скролл если чипов больше чем помещается
