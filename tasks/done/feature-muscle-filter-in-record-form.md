# Filter exercises by muscle group in record form

**Status**: DONE
**Priority**: P1
**Category**: ux

## Description
При записи подхода (ExerciseRecordForm, шаг pickExercise) упражнения фильтруются только по тексту. Нужно добавить фильтр по группам мышц — чипсы/теги мышц над списком упражнений. Пользователь нажимает "Chest" → видит только упражнения с грудными мышцами. Это критично когда упражнений много (50+).

## Part 1: Muscle filter in record form
### Current Flow
1. Пользователь видит текстовый Input для поиска
2. Ниже — список всех упражнений (filteredExercises)
3. Фильтрация только по `name.toLowerCase().includes(query)`

## Target Flow
1. В шаге pickExercise над текстовым поиском — компактная карта мышц (MusclesMap / MusclesMapFemale в зависимости от настройки)
2. Пользователь кликает на мышцы на карте — они подсвечиваются как фильтр
3. Список упражнений фильтруется: текстовый запрос AND (любая из выбранных мышц совпадает с exercise.muscles — OR логика)
4. Повторный клик на мышцу снимает фильтр. Если ни одна мышца не выбрана — показываются все упражнения.

## Files to Change
- `frontend/src/components/ExerciseRecordForm/index.tsx` — добавить MusclesMap + состояние фильтра мышц в step `pickExercise`
- `frontend/src/components/ExerciseRecordForm/styles.module.css` — стили для карты (компактная высота)

## Technical Notes
- MusclesMap уже поддерживает `onMuscleClicked` с toggle — тот же паттерн что в ExerciseForm
- Нужно читать `storage.get<string>('mapSex', 'male')` для выбора мужской/женской карты
- Карта должна быть компактной (~150-200px) чтобы список упражнений был виден
- Подсвеченные мышцы = текущий фильтр (массив Muscles[])

## Part 2: Shortcut — клик по мышце на Home открывает форму с фильтром

### Idea
На Home странице карта мышц сейчас декоративная (onMuscleClicked = noop). Нужно сделать её интерактивной: клик по мышце → открывается модалка добавления упражнения (ExerciseCreate) с предустановленным фильтром по этой мышце.

### Flow
1. Пользователь на Home видит карту мышц
2. Кликает на бицепс → открывается модалка добавления записи
3. В модалке уже выбрана мышца "Bicep" как фильтр → список показывает только упражнения на бицепс
4. Пользователь выбирает упражнение → заполняет подход → Save

### Files to Change

- `frontend/src/components/ExerciseRecordForm/index.tsx` — принимать опциональный проп `initialMuscleFilter?: Muscles[]` для предустановки фильтра
