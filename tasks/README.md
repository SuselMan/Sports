# Tasks

Папка для задач проекта KekTrack. Каждая задача — отдельный .md файл.

## Naming Convention
`NNN-short-description.md` — где NNN порядковый номер.

## Task Template
```markdown
# Task Title

**Status**: TODO | IN_PROGRESS | DONE
**Priority**: P0 | P1 | P2
**Category**: security | ux | feature | infra | quality

## Description
Что нужно сделать и зачем.

## Acceptance Criteria
- [ ] Критерий 1
- [ ] Критерий 2

## Notes
Дополнительные заметки.
```
